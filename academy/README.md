# Azure Architect Re-Entry Lab

Course promise: rebuild Azure engineering fluency at architecture depth.

This academy track is for experienced architects, solution architects, consultants, and design authorities who understand architecture but need to reconnect with the current Azure engineering surface.

Velocity Academy is the public learning experience. This repository is the Markdown source of truth for the Azure course, labs, notes, hints, artefacts, testing, and sign-off model.

## Tracks

| Track | Outcome |
|---|---|
| Foundation | Understand the Azure pattern. |
| Practitioner | Complete the lab and produce the artefact. |
| Senior Architect | Review, challenge, govern, and translate the pattern into engineer-ready design. |
